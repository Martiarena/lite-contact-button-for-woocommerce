Lightweight WhatsApp contact buttons for WooCommerce products and pages.

== Description ==
Lightweight contact button for WooCommerce products, with WhatsApp integration and no unnecessary scripts.

== Features ==
* Floating WhatsApp button
* WooCommerce single product button
* Supports variable products
* Elementor compatible (shortcode)
* Lightweight and modular

== Installation ==
1. Upload the plugin
2. Activate it
3. Configure it under "WhatsApp Buttons"

== FAQ ==
= Does it work without Elementor? =
Yes. The WooCommerce button appears automatically above the Add to Cart button.
